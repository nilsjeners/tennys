# TENNYS Design Guidelines

- Fonts: Barlow Semi Condensed (text), Barlow Condensed (numbers).
- Primary: #0b1d41 for headers and key UI chrome.
- Secondary: #064e3b for performance/progress accents.
- Accents: #22d3ee for primary actions; #84cc16 for achievements.
- Neutrals: #f9fafb background, #111827 primary text, #334155 secondary text, #e5e7eb dividers.
- Status: success #16a34a, warning #f59e0b, error #dc2626.
- Usage: deep blue dominant, green sparingly, cyan for CTAs, backgrounds mostly off-white or charcoal.
